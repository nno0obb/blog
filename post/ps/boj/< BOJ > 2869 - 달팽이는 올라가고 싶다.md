- BOJ :: [2869 - 달팽이는 올라가고 싶다](https://www.acmicpc.net/problem/2869)

---

```python
"""  
# BOJ  
# No. 2869  
# Python 3.10.4  
# by "nno0obb"  
"""  
  
  
def main():  
    # Input  
    A, B, V = map(int, input().split())  
  
    # Logic  
    ans = ((V-A) + (A-B-1)) // (A-B) + 1  
  
    # Output  
    print(ans)  
  
  
if __name__ == '__main__':  
    main()

```
